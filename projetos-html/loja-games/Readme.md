# Loja de Games

Projeto desenvolvido em HTML simulando uma loja de jogos.

## Funcionalidades
- Exibição de produtos em tabela
- Informações como nome, plataforma, preço e estoque
- Destaque para produto mais vendido
- Lista dos jogos mais populares
- Lista de plataformas disponíveis

## Tecnologias utilizadas
- HTML

## Objetivo
Praticar a criação de tabelas, listas e estruturação de páginas web.

## Observações
Este projeto faz parte dos meus estudos em desenvolvimento web e está em constante evolução.
