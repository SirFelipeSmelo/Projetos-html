# Campeonato de Futebol 2026

Projeto desenvolvido em HTML simulando uma tabela de resultados de um campeonato de futebol.

## Funcionalidades
- Tabela com classificação dos times
- Informações de jogos, vitórias, empates, derrotas e pontos
- Destaque para o time campeão
- Lista dos 3 primeiros colocados
- Lista de times participantes
- Link externo para site oficial
- Navegação interna na página

## Tecnologias utilizadas
- HTML

## Objetivo
Praticar a criação de tabelas, uso de listas, links e estruturação de páginas web.
